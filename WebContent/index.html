Proposta

O projeto busca a integração dos canais de autoatendimento, aumentando a eficiência na implementação e manutenção dos canais. A ideia consiste em criar um back-end genérico, capaz de abstrair as especificidades negociais de cada canal através de uma API REST de acesso comum.

Especialização

O fim do acoplamentos entre frontend e backend vai permitir especialização de equipes, aumentando o foco nas necessidades do canal

Escalabilidade

A esturura de descentralizada garante a independencia entre os componentes, garantindo a escalabilidade em nível de infra e possibilitando o desenvolvimento independente entre as equipes participantes.

Economia

Com a implantação dessa solução de infraestrutura, prevê-se uma economia de 40% no tempo de desenvolvimento/programação de novas soluções para os canais APF, APJ, Mobile e TAA.
